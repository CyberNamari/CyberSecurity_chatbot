# CyberSecurity Chatbot

A modular C# console chatbot for cybersecurity education.

## Features
- ASCII logo rendering
- Sound playback at startup
- Security questions for user checkup
- Free-form keyword chat

## How to Run
1. Clone the repository
2. Open in Visual Studio
3. Build and run

## Multimedia
- `Sound.wav` plays at startup
- `security.jpg` is converted to ASCII art
